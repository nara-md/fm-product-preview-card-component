# Frontend Mentor - Product Preview Card Component

A clean, responsive implementation of the [Frontend Mentor Product preview card component challenge](https://www.frontendmentor.io/challenges/product-preview-card-component-GO7UmttRfa).

## Features

- **Semantic HTML5:** Built with semantic markup (`<main>`, `<article>`, `<footer>`).
- **Responsive Design:** Mobile-first layout that looks great on all screens.
- **Modern CSS:** Clean variables, reset guidelines, and flexible layout techniques (Flexbox and Grid) for centering and alignment.
- **Aesthetic Fonts:** Custom integration of Google Fonts (Montserrat and Fraunces).

## Built With

- Semantic HTML5 Markup
- CSS Custom Properties (Variables)
- Flexbox and Grid Layout
- Google Fonts (Montserrat & Fraunces)

## Getting Started

To run or view this project locally, simply:

1. Clone this repository:
   ```bash
   git clone https://github.com/nara-md/fm-product-preview-card-component.git
   ```
2. Open [index.html](index.html) in your browser of choice.
