## My process

### Built with

- Semantic HTML5 markup
- CSS Custom Properties (Design Tokens)
- Flexbox / CSS Grid
- Mobile-first workflow
- Clean, modern CSS Reset

### What I learned

- Implementing modern CSS Reset principles like `min-height: 100dvh` for mobile viewports, `box-sizing: border-box`, responsive image behavior, and reduced-motion handling.
- Structuring layout code with CSS variables to handle design typography (Fraunces & Montserrat fonts) and colors smoothly.
- Handling responsive image switching cleanly via HTML5 `<picture>` and media queries.

### Key Features

1. **Modern CSS Reset**: A tailored setup using `reset.css` that standardizes font sizes, borders, margin/padding behavior, and responsive media rules.
2. **Flexible & Accessible layout**: Uses grid centering on the `body` element to perfectly center the product card and attribution footer on the screen, matching viewport heights correctly.
3. **Hover states**: Dynamic transition effects on buttons (e.g. background changes from Green 500 to Green 700).
